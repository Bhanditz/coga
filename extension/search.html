Provide a search capability

Search capability enables user to find the content they are looking for.

Suggestion for Priority Level: AAA

Related Glossary additions or changes

none

Principle and Guideline

Principle 3, Guideline 3.1

Description

Search capabilities enables a user to conveniently find and access information.

Benefits

Search capabilities enables a user to conveniently find and access information.

Related Resources (optional)

none

Testability

Procedure

  1. Check that a search box exist.
  2. Search box should be placed in a prominent position on the page and be visible across the entire site, ideally near the top navigation.
  3. Exception is provided were adding a search box is a significant expense for the site. An exception is provided for small organizations and organizations with small websites that includes simple links to each page from the home page . Note: A small organization is defined as not more than 50 employees, or a balance sheet total of less than 3.26 million, or the definition within your jurisdiction.

Expected Results

  • All checks above are true

Techniques

  1. Making a search box available.
  2. Positioning search box in a prominent position and visable across entire site.
