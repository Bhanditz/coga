Finding Help

Finding help is intended to provide assistance to users.

Suggestion for Priority Level: AAA

Related Glossary additions or changes

none

Principle and Guideline

Principle 3, Guideline 3.1

Description

Finding help enables a user to conveniently access information needed to understand how to use the website effectively.

Benefits

Finding help provides quick answers to users questions.

Related Resources (optional)

none

Testability

Procedure

  1. Check that help content, support page or support function exist and is reachable with one user action.
  2. When human help is available the correct contact information or mechanism should be reachable within two user interactions.

Expected Results

  • All checks above are true

Techniques

  1. Making the help content reachable with one user action.
  2. Making the human help reachable within two user interaction.
  3. Making help short, descriptive explanation of each topic that can be read quickly. Users who come to help pages are usually already confused, so they aren’t inclined to read long blocks of text.
  4. Making link to help content in the header and footer, especially on long pages and the home page.
